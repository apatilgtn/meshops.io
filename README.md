# MeshOps.io

A professional service mesh operations and support platform.

## Setup

1. Clone the repository:
```bash
git clone <your-repo-url>
cd meshops.io
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
